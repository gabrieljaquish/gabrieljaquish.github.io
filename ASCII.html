ASCII Table

The American Standard Code for Information Interchange or ASCII was developed from standard codes used in Telegraphs who's standard was first published in 1963 by the American Standards Organization (ASA), which later became the American National Standards Institute (ANSI). ASCII is a single byte character encoding only using the bottom seven bits
